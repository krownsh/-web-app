
# 變更日誌

## [2025-12-23] 首頁排版調整與視覺優化

### 功能優化
1.  **首頁 (HomeScreen)**:
    *   **Hero 區視覺調整**: 背景色改為主視覺「苔蘚綠 (zen-moss)」，並優化漸層效果，使文字更加清晰且符合整體設計風格。
    *   **版面重構 (Refactor Layout)**: 將原本散落的功能區塊（快速功能、必買清單）統一封裝進一致的「卡片式」容器中。
        *   **快速功能 (Quick Actions)**: 新增標題區塊，並將按鈕群組放入白色半透明卡片中。
        *   **必買清單 (Must Buy List)**: 將清單內容與新增按鈕統一放入卡片區塊，視覺上更為整潔且層次分明。
    *   **一致性**: 所有主要功能區塊現在都有統一的標題樣式（左側帶有顏色點綴）與圓角卡片背景，提升整體 UI 的協調性與易讀性。
